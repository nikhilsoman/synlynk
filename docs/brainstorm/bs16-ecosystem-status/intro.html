BS-16: Agent Compatibility Scorecard

Brainstorming session — attach rate · functional compat · interop scores

We're designing a scoring system that answers three questions about every agent synlynk can work with:

  • Attach rate — is the agent installed and reachable?
  • Functional compat score — of synlynk's known verb surface, how many work on this agent?
  • Interop score — when Agent A dispatches to Agent B, what's the cross-agent compatibility?
