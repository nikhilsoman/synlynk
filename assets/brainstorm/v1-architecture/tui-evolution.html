Full admin console. Org hierarchy, SSO/IdP integration, RBAC policy editor, compliance dashboard, multi-region room topology. Deployed as a standalone service.

Enterprise additions vs Team portal
  • Org hierarchy tree (org → team → project → agent)
  • SSO/IdP provider config (Okta, Azure AD, Google Workspace)
  • RBAC policy editor — visual rule builder, not YAML
  • Competency threshold configuration per org policy
  • Immutable audit log viewer with retention controls
  • Multi-region NATS cluster topology map
  • Integration marketplace (Jira, Linear, Slack, PagerDuty)
  • Trust score override + appeals workflow
  • Air-gap deployment mode (no external calls)
What stays the same across all tiers
  • Information architecture (same five core panels)
  • Room/message model (subject namespace, msgpack payload)
  • Competency vector schema (scores just gain more signal sources)
  • Agent instruction file editor (CLAUDE/GEMINI/AGENTS/AGY)
  • Board view (GH Projects v2 → Jira/Linear at enterprise)
  • Memory browser (project-docs → room history)
  • The concept of "funding" an agent via room subscriptions
