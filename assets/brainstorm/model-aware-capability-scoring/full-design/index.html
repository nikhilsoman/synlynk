Model inheritance — replacing the config file

Instead of hardcoded model_inheritance.json multipliers, Tokq publishes a model family registry — a public Memory Unit at tokq://registry/model-families. synlynk fetches it on upgrade. Contains family lineage + capability delta estimates per domain. Local fallback: neutral multiplier (1.0) until 3 real samples exist.

Split-model story: if model_at_dispatch ≠ model_at_completion, the rating row records both and sets a split_model = true flag. Routing excludes split-model stories from score aggregation (ambiguous attribution). Tokq exposes them separately as "mixed-model execution" samples — lower confidence, lower value, but still informative.
