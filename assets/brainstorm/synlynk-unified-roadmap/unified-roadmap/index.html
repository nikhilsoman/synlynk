🤝 Coexist, don't compete

Every tool in the ecosystem is a distribution channel. A SuperPowers user who hits the "no persistent memory" ceiling is the ideal Synlynk prospect. Help them — don't fight them.

📐 Open protocol, not lock-in

The context protocol is two commands any tool can call. It should be documented, stable, and boring. The value is in the state Synlynk maintains — not in making integration hard.

⚡ Genuine value first, features second

Each milestone ships one thing developers will feel immediately: persistent memory, coordinated routing, cross-machine sync, marketplace knowledge. No milestone is about internal plumbing alone.
