synlynk — VPS Deep-Dive

Relay hosting for v0.9.3 workgroup relay · WSS/443 · stateless · 4–20 devs

Key finding: There is no purpose-built "agentic relay" platform yet. "AI-native hosting" in 2025–2026 mostly means GPU inference hosting, not persistent-process agent coordination. Fly.io remains the best fit for stateless WSS relay hosting. Bedrock AgentCore and Azure Foundry are the closest to agentic infra — but are designed for agent session management, not relay routing.

Top 3 Profiles

🥇 Recommended
Fly.io
$3–5 / mo · shared-cpu-1x · 512MB
  • WSS on 443 — trivial config
  • Global Anycast, 35+ regions
  • pin min_machines_running=1 to stay warm
  • Scale to zero → warm → global = one config
  • Best migration path: LAN → CF Tunnel → Fly.io (same URL)
  • Synlynk can publish a ready-made fly.toml
🥈 If BYOC/Team-Isolation Needed
Northflank
$4–8 / mo · per-second billing
  • BYOC: run on your own AWS/GCP
  • Persistent containers + scheduled jobs on same platform
  • Team/org workspace model — good for community relay
  • More complex setup than Fly.io
  • Better fit for v1.1+ community server than v0.9.3 relay
🥉 Budget Pick
Hetzner CX11
€4.51 / mo · 2 vCPU · 2GB RAM
  • Massively overspecced for 512MB relay
  • You own nginx + Let's Encrypt + systemd
  • EU-centric (Nuremberg, Falkenstein, Helsinki, Ashburn)
  • Zero managed features — all DevOps on you
  • Right if: team has ops expertise + cost matters most

Ownership Model — Which path for synlynk?

A
Each team self-provisions
Every workgroup spins up their own relay. synlynk ships a fly.toml + one-command deploy. You never run infra.
→ Best VPS: Fly.io (user account, their $3–5/mo)
⚠️ Fragmented: no network effect across teams, no "find a public relay"
B
synlynk hosts community relay
synlynk.com runs a shared relay. Teams register, get a namespace. Free tier; scale as community grows.
→ Best VPS: Northflank (team isolation) or Fly.io global (cost)
⚠️ You run infra. Needs auth + namespace isolation. Best post-v1.0.
C
Hybrid — both options
v0.9.3: self-provision (ship fly.toml). v1.1: synlynk community relay as upgrade path. Teams start free, graduate to self-hosted.
→ v0.9.3: Fly.io self-host · v1.1+: synlynk Northflank/Fly.io
✅ Natural graduation: LAN → CF Tunnel → self-VPS → community server
