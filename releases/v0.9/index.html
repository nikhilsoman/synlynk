Context / Dispatch / Relay

curl -sSL https://raw.githubusercontent.com/nikhilsoman/synlynk/main/install.sh | bash

What's new

  • Scoped dispatch context — Task-level context slice replaces 7-day devlog dump in every agent dispatch
  • Dispatch quality enhancements — ## Relevant Files, ## How to Verify, and per-agent prompt framing in every dispatch
  • Ed25519 capability signing — Every capability rating is signed with a local Ed25519 key — tamper-evident from day one
  • Async daemon — Always-on process — agents pull context over HTTP without invoking the CLI

Changelog

  • v0.9.4 Jun 24, 2026 SQLite-primary task state with generated todo.md, per-agent context profiles (.agents/<agent>.json), synlynk jobs SQLite read + --watch, pre-flight dispatch gate, HTTP SSE relay broker (synlynk relay start/broadcast, port 27472), VERIFY_SKIP sentinel pattern, dispatch_flags scoping for --dangerously-skip-permissions; 472 tests
  • v0.9.3 Jun 23, 2026 synlynk daemon — always-on process: double-fork daemonization, HTTP context server on localhost:27471 (10 endpoints), persistent daemon_jobs queue with priority + dependency chains, zombie-safe reaping via waitpid, per-job commit for crash-safe restarts, --install-service for launchd/systemd/crontab; 432 tests
  • v0.9.2 Jun 22, 2026 synlynk join (new member onboarding: devlog seed from git history, AI context file regeneration, team digest), synlynk team status (active members, story assignments, token budgets), synlynk decide (multi-agent consensus panel with signed Decision records), pull-before-write arbitration, --tokens flag on story create; 390 tests
  • v0.9.1 Jun 22, 2026 Install hardening: installed binary now embeds synlynk/ package so it works outside the source repo. --docs-dir flag on init for repos with non-standard doc locations. Smart init migration: detects existing docs and migrates rather than overwriting with blank templates
  • v0.9.0 Jun 21, 2026 Scoped dispatch context (task-level instead of full devlog), ## Relevant Files injection from source-map, ## How to Verify contract per dispatch, per-agent framing (Codex/AGY/Claude), Ed25519 capability rating signing, anti-gaming quality cap (test_count<3 capped at 5.0), synlynk/ package split; 365 tests
