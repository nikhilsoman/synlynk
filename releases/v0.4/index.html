What's new

  • Agent discovery — init wizard scans for Claude, Gemini, Codex, and AGY — shows your workgroup at a glance
  • Background dispatch — Fire-and-forget agent jobs from any shell — synlynk dispatch claude --task "..."
  • Parallel trio dispatch — synlynk run --trio dispatches one task to all functional agents simultaneously
