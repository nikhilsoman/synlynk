What's new

  • Context injection — Reads project-docs/ into .synlynk/context.md before each exec
  • Flatline detection — Alerts and halts when the last 3 runs share identical command + non-zero exit
  • Cost tracking — Appends per-session token counts and $ cost to project-docs/costs.md
  • Budget enforcement — Blocks exec when cumulative cost or request count exceeds configured limits
  • Telemetry log — Rolling 100-entry log of every exec — duration, exit code, cost, command
