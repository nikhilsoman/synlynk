Dispatch quality enhancements

## Relevant Files, ## How to Verify, and per-agent prompt framing in every dispatch

What it does

v0.9.0 adds three quality layers to every agent dispatch. `_relevant_files_for_story()` injects a `## Relevant Files` section (up to 10 paths from the source-map, filtered by engineering domain) so agents know exactly which files to touch. `_verify_contract_for_story()` injects a `## How to Verify` section with a concrete pytest invocation derived from the story title. `_format_prompt_for_agent()` applies agent-specific framing: Codex receives `## Task Criteria` as scannable bullets; AGY receives a concise `Task:` prefix with 2000-char context; Claude receives the full narrative prompt.

Usage

$ synlynk dispatch codex --story story-abc123 --task "fix token extraction" Codex receives Task Criteria bullets + Relevant Files list + How to Verify pytest command
