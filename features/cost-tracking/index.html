Cost tracking

Appends per-session token counts and $ cost to project-docs/costs.md

What it does

After each `synlynk exec` run, the harness scrapes token counts from captured AI CLI stdout using format-specific regex patterns, computes cost at hardcoded rates ($0.003/1K input + $0.015/1K output), appends a row to `project-docs/costs.md`, and prints a Budget Pulse summary to the terminal.
