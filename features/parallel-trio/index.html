Parallel trio dispatch

synlynk run --trio dispatches one task to all functional agents simultaneously

What it does

`synlynk run --trio <task>` dispatches the same task description to every functional agent (claude, agy, codex — whichever are installed) in parallel background jobs. Each agent runs independently; `synlynk jobs` shows all three. This is the first form of horizontal parallelism in synlynk: same task, multiple perspectives, results streamed to separate log files for human review.

OS Layer

IPC — Agent Files & Coordination

Introduced in

v0.4.x — Hybrid Workgroup Bootstrap

Usage

$ synlynk run --trio "write unit tests for src/auth.py" Dispatches to all functional agents in parallel; each gets its own job ID
