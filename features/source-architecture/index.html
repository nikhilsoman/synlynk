Source architecture injection

## Source Architecture snapshot injected into every exec context from a passive git-HEAD cache

What it does

v0.7.0 adds a language-agnostic static scanner that extracts symbols from up to 9 languages (Python, JS, TS, Go, Rust, Ruby, Java, Kotlin, Shell) and injects a `## Source Architecture` section into every `synlynk exec` context. The scan cache is keyed by git HEAD SHA — zero overhead on cache hit. A file scoring formula (entry-point bonus, git frequency, depth penalty) surfaces the 15 most relevant files. Full results are stored in the `source_symbols` SQLite table and `project-docs/source-map.md`. `synlynk scan [--deep] [--status]` provides manual control.

Usage

$ synlynk scan Scans source tree and injects ## Source Architecture into context; zero cost on cache hit
$ synlynk scan --deep Full scan: writes source_symbols to state.db and source-map.md
$ synlynk scan --status Shows cache age, git HEAD SHA, and file/symbol counts
